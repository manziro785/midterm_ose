Описать базовый процесс работы с git.

Ну сначало для создания файла используем    mkdir название файла
Для того чтобы войти в файл                 cd название файла
для того чтобы добавить все изменения в индекс git add . или git add название файла
для того чтобы закоммитить                  git commit -m 'название коммита'
Создание git репозитория                    git init
